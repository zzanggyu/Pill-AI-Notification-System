# 알약 정보 알림e (Pill-AI-Notification-System)
알약 정보를 쉽고 빠르게 확인할 수 있는 AI 기반 의약품 정보 제공 서비스

## 📋 프로젝트 소개
알약 정보 알림e는 AI 딥러닝 기술을 활용하여 의약품 이미지를 인식하고, 해당 의약품의 정보를 즉시 제공하는 모바일 애플리케이션입니다. 사용자는 알약의 사진만 찍으면 효능, 용법, 주의사항 등 필요한 정보를 쉽게 확인할 수 있습니다.

### 기획 배경
- 국민 6명 중 1명이 약물 오남용 경험
- 65세 이상 44%가 매일 5개 이상 약 복용
- 의약품 혼용 금지 조합 및 부작용에 대한 정보 접근성 부족
- 처방약 정보를 잊어버리거나 출처가 불분명한 약에 대한 정보 필요

## 📊 요구사항 분석

### 설문조사 개요
- 총 응답자 수: 217명
- 조사 항목: 약의 정보 필요성, 앱 사용 의향, 주요 사용 목적, 선호 기능 등

### 주요 분석 결과

#### 1. 약의 정보 필요성
- 약의 정보가 궁금했거나 효과를 잊어버린 경험
  - 있음: 158명 (73%)
  - 없음: 59명 (27%)
- 높은 비율의 사용자가 약 정보 조회 필요성을 느낌

#### 2. 앱 사용 의향
- '의약품 알림e' 앱 사용 의향
  - 있음: 172명 (79%)
  - 없음: 45명 (21%)
- 대다수의 응답자가 앱 사용에 긍정적인 반응

#### 3. 주요 사용 목적
- 복용하는 약의 정보가 궁금해서: 136명 (45%)
- 여러 약의 혼용 여부가 궁금해서: 109명 (36%)
- 약의 정보를 읽기 힘들어서: 36명 (12%)
- 타인이 복용하는 약을 관리해서: 12명 (4%)
- 기타: 6명 (2%)
  - 성명서나 약 박스를 버리는 경우 참고
  - 비상한 중상인 사람이 복용을 해도 괜찮은지 궁금
  - 약의 정확한 정보를 알고 싶어서
  - 부작용이 궁금
  - 약력을 시간을 정하기 위해
  - 지인소개

#### 4. 선호 기능
- 알약 정보 제공 기능: 153명 (34%)
- 알약 혼용 여부 제공 기능: 153명 (34%)
- 알약 복용 리마인더(복용 알림): 71명 (16%)
- 음성으로 정보를 제공하는 기능(TTS): 43명 (9%)
- 알약 개수 카운팅 기능: 27명 (6%)
- 기타: 6명 (1%)
